How It Works

GitHub Actions runs daily at 07:00 Dubai time, crawling all ~2,600 PropertyFinder listings for Al Hamra Village across ~110 pages.

Each listing price is compared against the previous day snapshot. Any decrease is flagged as a price drop.

All data is static JSON served via GitHub Pages. No backend. No cookies. 🔒
