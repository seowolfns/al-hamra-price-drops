# Al Hamra Village Price Drop Monitor

Daily price drop tracker for Al Hamra Village, Ras Al Khaimah.

**Live site:** https://seowolfns.github.io/al-hamra-price-drops/

Powered by GitHub Actions + GitHub Pages. Crawls ~2,600 PropertyFinder listings daily.

## Setup
1. Settings → Pages → main / root → Save
2. Actions → Crawl PropertyFinder & Deploy → Run workflow (baseline)
3. Run again next day to see first drops
